# FAERS-Pharmacovigilance-Pipeline
This repository contains the final capstone project for Data Engineering Zoomcamp 2026 hosted by Data Talks Club. This project uses the FDA's Adverse Event Reporting System (FAERS) as the data source and create a final dashboard to interactively visualize the data.

**Note: Generative AI services like Calude Code and ChatGPT was used for assistance in creating this project, with strict human supervision**
